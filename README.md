
Telegram-бот с картами Таро на основе героев Dota 2.

## Возможности
- Карта дня с предсказанием и советом по герою
- Коллекция выпавших карт
- Ежедневная рассылка в полночь
- 127 героев с уникальными текстами

## Стек
Python • aiogram 3 • aiosqlite • OpenDota API
